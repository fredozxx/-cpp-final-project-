# Crypto Password Generator

A C++ console application that generates a strong, random password and saves it to a file.

### Features:
1. **Strong Password Generation**: Generates a password with mixed characters.
2. **File Handling**: Saves the generated password to a file.
3. **Input Validation**: Ensures that the user inputs a valid password length.
